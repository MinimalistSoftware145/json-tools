# JSON Tools

Make working with JSON files **WAY** easier!

### We have 2 scripts here.

1. JSON Reader
Parses JSON files and makes them show up in a human-readable format!
Even your *grandma* can read JSON now.
2. JSON Maker
Why write JSON code? Now you can input key name, value, click Make, repeat,
then input the path, and click Save!
Saving involves robust error handling where it will not crash on errors -
it will show the error!

### Dependencies
Python.
